# outlier
AI Running Coach

I wanted to create an AI running coach which uses my Strava activities, local weather data and connection to LLM to provide useful insights on my runs.

Planned Features:
- Strava API integration
- Weather API integration
- Database storage
- AI training insights
- Telegram bot interface

Tech Stack:
- Python
- Strava, Weather, OpenAI, Telegram API
- Supabase
